Redefinindo as propriedades padrões dos navagadores.

Reset CSS é uma folha de estilos que irá sobrepor todas as propriedades de estilo CSS que os navegadores colocam por padrão aos elementos, ajudando com que sites tenham o mesmo design em todos os navegadores.

Exemplo:

Título 1

Título 2

Título 3

  • Item 1
  • Item 2
  • Item 3

Esse é um parágrafo contendo formatação como negrito, itálico e riscado
