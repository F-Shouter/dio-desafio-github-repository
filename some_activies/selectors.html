Welcome!

  • tipo:busca elementos por um tag HTML específica
  • id (#):busca elementos através do atributo "id", id é único para um elemento dentro da mesma página HTML. Para cada página, ele não pode se repetir.
  • classes (.):busca elementos através do atributo "class"
  • universal (*):seleciona todos os elementos HTML
  • atributo (atribut):seleciona elementos que possuem um atributo específico em nosso documento HTML e conseguimos também ar atributos com um valor específico.
