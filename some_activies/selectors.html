  • tipo:busca elementos por um tag HTML específica

  • id (#):busca elementos através do atributo "id", id é único para um elemento dentro da mesma página HTML. Para cada página, ele não pode se repetir. id não pode ter espaços dentro das aspas!

  • classes (.):busca elementos através do atributo "class". Classe serve para identificar um grupo de elementos, então podemos dar estilos para vários elementos de uma vez só.

  • universal (*):seleciona todos os elementos HTML

  • atributo [(atributo)]:seleciona elementos que possuem um atributo específico em nosso documento HTML e conseguimos também ar atributos com um valor específico.

    • [atributo=valor], o valor tem que ser exatamente igual!
    • [atributo~=valorX], o valorX vai ser procurado nos titles, ou o valorX exato ou separado por espaço! O valorX tem que ser isolado no atributo!
    • [atributo|=valorY], funciona com o valorY só, ou com o valorY-valorZ. Porém o valorY tem que ser o primeiro! e grudado com o ifem!
    • [atributo*=valor]vai procurar o valor independente do lugar, por exemplo dentro de um link.

    • PREFIXO: o começo do valor precisa estar entre aspas!
      • [atributo^=começo do valor]

    • SULFIXO o fim do valor precisa estar entre aspas! ele consegue pegar independente se estiver separado com pontos ou barras.
      • [atributo$=fim do valor]
